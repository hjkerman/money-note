# 테스트 절차

이 문서는 `money-note`를 실제로 써보기 전에 확인할 항목을 정리한다.

## 1. 서버와 로그인

1. Docker 서버를 실행한다.

```bash
docker compose up -d --build
```

2. health를 확인한다.

```bash
curl http://localhost:18080/health
```

기대값:

```json
{"status":"ok"}
```

3. 웹 앱 `http://localhost:5173`에 접속해 로그인한다.

확인할 것:

- 로그인 전에는 조작 화면이 보이지 않는다.
- 로그인 후 요약과 당월 기록이 표시된다.

## 2. 당월 지출 입력

1. `당월 > 당월 지출` 탭을 연다.
2. 날짜, 사용처, 사용항목, 금액을 입력한다.
3. `추가`를 누른다.

확인할 것:

- 새 행이 당월 지출 테이블에 나타난다.
- 적요는 `[사용처] 사용항목` 형식으로 보인다.
- 요약의 카드대금이 증가한다.

## 3. 분류 변경과 저장

1. 당월 지출의 `분류` select를 바꾼다.
2. 상단에 `변경 사항 저장` 버튼이 뜨는지 확인한다.
3. 아직 새 항목을 추가하지 말고 `새로고침`을 눌러도 된다. 저장 전 변경이 화면에 어떻게 보이는지 확인한다.
4. `변경 사항 저장`을 누른다.

확인할 것:

- 분류 변경만으로 즉시 서버 저장되지는 않는다.
- 저장 버튼을 누르면 상태 문구가 저장 완료로 바뀐다.
- 새로고침 후에도 저장한 분류가 유지된다.

## 4. 청구와 타인정산 삭제/초기화

1. `당월 > 청구` 탭에서 테스트 항목을 하나 추가한다.
2. 해당 행의 `삭제`를 눌러 삭제한다.
3. 다시 여러 항목을 추가한 뒤 `초기화`를 누른다.
4. `타인정산`에서도 같은 절차를 반복한다.

확인할 것:

- 행별 삭제가 된다.
- 초기화는 해당 탭의 모든 행만 삭제한다.
- 당월 지출, 현금흐름, 할부 항목은 같이 삭제되지 않는다.

## 5. 동결 확인

1. `동결` 탭에 항목을 추가한다.
2. `확인`을 누른다.
3. confirm 창에서 승인한다.

확인할 것:

- 동결 항목이 사라진다.
- 같은 금액과 적요가 당월 지출에 추가된다.

## 6. 할부

1. `당월 > 할부` 탭을 연다.
2. 할부액 `100000`, 수수료율 `1.7`, 개월수 `3`을 입력한다.

확인할 것:

- 수수료는 `1,700원`이다.
- 월 납입액은 `(100000 + 1700) / 3`의 원 단위 올림인 `33,900원`이다.
- 카드대금 요약에 월 납입액이 반영된다.

## 7. 카드 정기결제

1. `고정지출 > 카드 정기결제` 영역을 확인한다.
2. 결제일, 사용처, 사용항목, 금액을 입력한다.
3. 추가된 항목의 `확인`을 누른다.

확인할 것:

- 카드 정기결제도 `[사용처] 사용항목` 형식으로 보인다.
- 확인하면 같은 사용처/사용항목을 가진 당월 지출이 생성된다.
- 카드 정기결제 항목은 삭제할 수 있다.

## 8. 통계와 월별 기록

1. 상단의 `통계 보기`를 누른다.

확인할 것:

- 소비 통계가 보인다.
- 월별 기록이 같은 영역 아래에 보인다.
- 큰 탭 묶음에는 `월별 기록`이 따로 나타나지 않는다.

## 9. 공유 페이지

1. 청구 공유 화면을 연다.

```text
http://localhost:18080/share/claim
```

2. 타인정산 공유 화면을 연다.

```text
http://localhost:18080/share/settlement
```

확인할 것:

- 로그인 없이 열 수 있다.
- 청구/타인정산 항목과 합계가 보인다.
- 계좌번호, 송금 링크, 개인정보는 보이지 않는다.
- 청구 화면에는 전월~당월 가계부 평가 한 줄이 보인다.

## 10. Excel export

1. 웹에서 `엑셀 export`를 누르거나 API를 호출한다.

```bash
curl -X POST http://localhost:18080/api/export
```

2. 최신 파일을 다운로드한다.

```bash
curl -O http://localhost:18080/api/export/latest.xlsx
```

확인할 것:

- 당월 기록의 새 지출이 `[사용처] 사용항목` 형식으로 export된다.
- 청구, 타인정산, 동결, 현금성 고정지출, 카드 정기결제 테이블 배치가 깨지지 않는다.
- 전체 기록의 과거 hard data가 보존된다.
